Azure Repository Pluginedit

The Azure Repository plugin adds support for using Azure Blob storage as a repository for Snapshot/Restore.

Installationedit

Version 7.13.0 of the Elastic Stack has not yet been released.

Removaledit

The plugin can be removed with the following command:

sudo bin/elasticsearch-plugin remove repository-azure

The node must be stopped before removing the plugin.
