Create a new packageedit

The elastic-package tool can be used to bootstrap a new package or add a data stream using an embedded archetype (resource templates). It’s advised to use elastic-package create to build new package rather than copying sources of an existing package. This will ensure that you’re following latest recommendations for the package format.

  1. Bootstrap a new package using the TUI wizard:

    elastic-package create package.

  2. Manually adjust the package:

    • define policy templates and inputs
    • add icons and screenshots
    • update README files
    • update changelog.yml file

  3. Verify the package

    Enter the package directory and ensure the package fits the spec:

    cd <your_new_package>
elastic-package check
